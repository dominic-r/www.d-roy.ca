Privacy Policy

Last updated: June 4, 2025

This Privacy Policy describes how I handle information when you use SDKO Web Services ("SWS"). SWS is my personal platform. For more information about what SWS is, please see the What is SWS section.

1. Information Collection and Use

I collect minimal information necessary to provide services:

  • Server logs containing access timestamps, IP addresses, and requested resources
  • Technical information necessary for service operation and security
  • Performance metrics for system optimization
  • Security event data for threat detection and prevention

No personal information is collected beyond what is required for basic service operation. I implement data minimization principles to ensure only necessary information is collected and stored.

All collected data is used solely for:

  • Ensuring system security and reliability
  • Troubleshooting technical issues
  • Improving performance and stability
  • Protecting against unauthorized access or attacks

2. No Tracking Policy

I do not implement any tracking or analytics on my public-facing websites, including:

  • *.sdko.org
  • *.sdko.net
  • *.cadoma.net
  • *.d-roy.ca

This means:

  • No cookies for tracking purposes
  • No JavaScript analytics tools
  • No user behavior tracking
  • No third-party tracking pixels or beacons
  • No fingerprinting technologies

Standard web server logs are maintained for security purposes only.

3. Data Storage and Location

All data is stored exclusively on servers located in Canada. For detailed information about server infrastructure, including the External Gateway Servers that handle external traffic, please visit the Infrastructure section of the SWS page.

By keeping all data in Canadian jurisdiction, I ensure:

  • Compliance with Canadian privacy expectations
  • Data is not subject to foreign jurisdiction laws
  • Lower latency for Canadian users
  • Geographic data sovereignty

4. Logging and Retention

I'm committed to privacy-focused practices. For detailed information about logging practices and security measures, please see the Logging and Data Retention section of the SWS page.

My retention policy follows these principles:

  • Data is kept only as long as necessary for its intended purpose
  • Regular purging of outdated information
  • Automatic deletion of logs after the retention period
  • Sensitive information is purged more frequently than standard operational data

5. Data Sharing

I do not sell, trade, or otherwise transfer information to outside parties. Data remains within Canadian infrastructure and is not shared with third parties except when required by law.

In the rare case where legal requirements mandate disclosure:

  • Only the minimum required information would be provided
  • Any such disclosure would be documented
  • Affected parties would be notified if legally permissible

6. Security

I implement appropriate security measures to protect information. For details on security measures, please see the Logging and Data Retention section of the SWS page.

My security approach is based on defense-in-depth principles, with multiple layers of protection including:

  • Network security controls
  • Encryption in transit and at rest
  • Access controls and authentication
  • Regular security patching
  • Intrusion detection and prevention

7. Contact Information

If you have important questions about this Privacy Policy, please contact me at: dominic@sdko.org

Please only use this contact for significant matters.
